Overview

A simple tool for manage datewise vendor payment planning. The employee will create the required payments in the draft state and that can be proceeded after approved by the higher authority

Configuration

No additional configuration is required.

Create a New Payment Proposal by navigating to the Invoicing Menu

Create a New Payment Proposal by navigating to the Invoicing Menu. Invoicing --> Payment Proposal --> Vendor Payment Proposal

Submitting a request

A employee can create a Bulk vendor payment record and submit it for approval from a higher authority

Approval

Higher authority should enter the approved amount and approve each record in the proposal line. and at last the proposal itself.

Payment

After the payments over. One can easliy record the details about the online payments and mark it as paid in the corresponding record.

Curresponding Vendor Payment is created.
